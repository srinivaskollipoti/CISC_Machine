Detailed Description

Enum type to distinguish the state of cpu. LOAD_MAR = phase to load MAR LOAD_MBR = phase to load MBR from memory LOAD_IR = phase to IR from MBR EXECUTE = phase to execute instruction INTERRUPT = phase indicating interruption NO_INST = phase indicating no more instruction
